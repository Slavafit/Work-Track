WorkTrack: license module notes

1. Active app files live in:
   app/src/main/java/com/example/worktrack/license/

2. Current license endpoint:
   https://license-server.slavafit.workers.dev

3. Current app id:
   worktrack

4. Android requirements:
   - INTERNET permission in AndroidManifest.xml
   - androidx.datastore:datastore-preferences dependency

5. Activation flow:
   - user enters email
   - app sends email, device_id, app_id to /activate
   - server may return active, trial, pending, or error
   - pending is shown as a normal status, not as a connection error

6. Do not keep duplicate license Kotlin files in the project root.
