Инструкция по интеграции лицензионного модуля
1. Скопировать файлы в app/src/main/java/.../license/ и поменять package на свой.
2. build.gradle:
kotlinimplementation "androidx.datastore:datastore-preferences:1.1.1"
3. AndroidManifest.xml:
xml<uses-permission android:name="android.permission.INTERNET" />
4. В LicenseManager.kt поменять две константы:
kotlinprivate const val BASE_URL = "https://license-server.slavafit.workers.dev"
private const val APP_ID   = "ИМЯ_ПРИЛОЖЕНИЯ"  // например: WorkTrack
5. В MainActivity.kt обернуть контент:
kotlinsetContent {
    AppTheme {
        LicenseGate {
            // существующий код без изменений
        }
    }
}
